# BilibiliTools chrome extension

## Features

- Get basic information of the current video (title, uploader, views, comments, etc.)
- One-Click Triple Action (Like, Coin, Favorite)
- Auto Comment Posting

---

## Installation (Local Use)

1. Download the repository code or `dist.zip` and unzip it.
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" at the top right.
4. Click "Load unpacked" and select the unzipped folder.
5. After the extension is loaded, click the toolbar icon to use it.

---

## Notes

- The extension only works on Bilibili web pages.
- To update features, repackage and replace the `dist` folder.
